# Pi Zero Blinkt

A customized rasbpian image which enables OTG ethernet features of the Pi Zero
and has Pimoroni Blinkt software preloaded

[Blinkt](https://learn.pimoroni.com/tutorial/sandyj/getting-started-with-blinkt)
